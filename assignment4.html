Challenge 5: Accessibility Concerns

Through these challenges, the greatest accessibility concerns presented themselves in challenge 2 because of the inherent probelm with tables. The necessary use of tables for this challenge created an unavoidable issue to viewers that may be physically impaired. The table element does not provide voice-over accessibility that allows visually-impaired persons to understand the table. To counter this problem, I would give each table element a header and description that makes sense to a screen reader. I would add a "scope" to the -th- elements to tell the screen reader what row and/column they are dealing with. In this particular case of Chess that would be adding column names A through H and row names 1 through 8, so the screen reader can read "White Queen to A9".
